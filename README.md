# Exercício

Lê uma lista de artigos devolvidos em formato JSON, no qual este mesmo é validado, e faz uma lista destes mesmos na qual é possível abri-los e ler o conteúdo. É possível atualizar a página no browser e manter-se no artigo.

## Como utilizar

1. Clonar o repositorio para um servidor(Apache ou Nginx)
2. Abrir em localhost/repositorio/index.html
